## Завдання 1
**Використайте popen(), щоб передати вивід команди rwho (команда UNIX) до more (команда UNIX) у програмі на C.**

Програма використовує дві функції `popen()`. Перша відкриває команду `rwho` для читання (щоб забрати її вивід), а друга - відкриває more для запису (щоб передати туди ці дані)
## Код програми знаходиться в файлі `magic_pipe.c`
```
nano magic_pipe.c
```
### Компіляція та запуск
Скомпілюємо програму звичним способом
```
gcc magic_pipe.c -o magic_pipe
```
За замовчуванням в Ubuntu Server команда `rwho` може бути не встановлена, або може не працювати без фонової служби (демона) `rwhod`. Тож її треба встановити
```
sudo apt update
sudo apt install rwho rwhod
sudo systemctl start rwhod
sudo systemctl enable rwhod
```
Після чого запускаємо програму звичним способом
```
./magic_pipe
```

## Завдання 8
**Напишіть програму, яка надає користувачу можливість видалити будь-який або всі файли у поточному робочому каталозі. Має з’являтися ім’я файлу з запитом, чи слід його видалити.**

Для видалення файлів у C використовується системний виклик `unlink()`. Він буквально відв'язує ім'я файлу від даних на диску, після чого файл зникає (якщо на нього немає інших жорстких посилань).
Про всяк випадок додамо прервірку, вона буде пропонувати видаляти лише звичайні файли (`DT_REG`) та ігноруватиме каталоги (включаючи `.` та `..`). 
Також додамо секретну клавішу `q (quit)`, щоб можна було перервати процес сканування, якщо раптом з видаленням передумали.

```

Він ідеально підходить для вимірювання інтервалів (як секундомір).

nano chronomancer.c
```
### Компіляція та запуск
```
gcc chronomancer.c -o chronomancer
```
P.S. На дуже старих системах Linux для функції `clock_gettime` іноді потрібно додавати прапорець `-lrt` в кінці

 (b) від 0.0 до n, де n — будь-яке дійсне число з плаваючою точкою.

gcc -Wall random_floats.c -o random_floats

До речі, прапорець `-Wall` означає "Warning All". Він змушує компілятор перевіряти код як під під мікроскопом і повідомляти про будь-які підозрілі місця.

3. Натискаємо 1, щоб побачити, що файл з'явився у системі.

4. Натискаємо 3, вводимо `spell.txt`, і ти бачимо свій текст, витягнутий прямо з "нутрощів" бінарника.

5. Натискаємо 4, щоб видалити файл, і знову перевіряємо список [1].
